# yokoffing/filterlists (AdBlock Plus Branch)

In this repository, you will find various Adblock Plus formatted filterlists, forked from [yokoffing's filterlist repository.](https://github.com/yokoffing/filterlists)

![1000004735](https://github.com/user-attachments/assets/5c77aa6c-8c67-4fe5-bda8-d000e95efc4a)

## Note
This forked branch was originally created to fulfill the lack of advanced filtering offered in Cromite's Android version, which until recently only had AdBlock Plus as a viable option for adblocking, however, it now supports installing Manifest V2 extensions, so the rules in this branch will likely be largely out of sync with the upstream offered by Yokoffing himself, as I now personally use the upstream branch.

## Advice
If you are a former user of Kiwi Browser looking for a multi-platform Chromium fork offering extension support, I recommend using Cromite and installing uBlock Origin; this will remove the need to use this branch entirely, however, I will try to update it occasionally going forward, so, do not consider this fork as abandoned.

You can install Cromite here:
https://github.com/uazo/cromite/releases
